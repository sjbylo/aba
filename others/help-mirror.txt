Install 'Mirror Registry for Red Hat OpenShift' (Quay) or connect to an existing mirror registry.

Usage:
  aba --dir mirror install [options]

Options:
  -d, --dir <path/to/dir>              # Run aba in the specified directory, or cd there and run without -d.
  -y                                   # Automatically accept the default answer to all prompts.
  -H, --mirror-hostname string         # The hostname of the mirror registry.
  -k, --reg-ssh-key ~/.ssh/mykey       # Path to ssh key, used to install Quay to a *remote host*.
  -U, --reg-ssh-user <string>          # Username to use to access the remote host via ssh.
  -u, --reg-user string                # The username used to access the registry (default is 'init').
      --data-dir </path/to/dir>        # Install Quay images/cache to this dir instead of ~/quay-install.
      --reg-path <string>              # The path to store images inside the mirror registry.

Related commands:
  aba [-d mirror] sync                 # Copy images from the Internet into the mirror registry (mirror2mirror).
    -r, --retry [<count>]              # Retry <count> times on oc-mirror error.

  aba [-d mirror] save                 # Copy images from the Internet to disk (mirror2disk).
    -r, --retry [<count>]              # Retry <count> times on oc-mirror error.

  aba [-d mirror] load                 # Copy images from disk to the mirror registry (disk2mirror).
    -r, --retry [<count>]              # Retry <count> times on oc-mirror error.

  aba [-d mirror] verify               # Verify mirror registry connection credentials are working.

Examples of fully disconnected mode (air-gapped):

  Configure mirror registry on *localhost* (ensure 'registry.example.com' points to the mirror registry):
    aba [-d mirror] install -H registry.example.com 

  Configure mirror registry on *remote host* with 'registry.example.com' using 'remote-ssh-user':
    aba [-d mirror] install -H registry.example.com -k ~/.ssh/id_rsa -U remote-ssh-user

  Configure mirror registry and use path ocp4/openshift4 and store all data under /mnt/large/drive/my-data
    aba [-d mirror] install -H registry.example.com -k ~/.ssh/id_rsa -U remote-ssh-user --reg-path ocp4/openshift4 --data-dir /mnt/large/drive/my-data

  Configure mirror registry on localhost with 'registry.example.com' and load the (previously saved) images to the mirror:
    aba [-d mirror] load -H registry.example.com  --retry 8

  Verify connection to mirror registry. First, install Quay or manually set up pull secret & CA cert files in aba/mirror/regcreds dir.
    aba [-d mirror] verify

  Install OpenShift:
    See: aba cluster --help

The 'install' command can be substituted with save, load, or sync. When these commands are used, 'install' will be triggered automatically if required.
The options above [-k, -U, -H etc] can be used also for 'save', 'load' and/or 'sync' and will be passed to the 'install' command, if needed.
